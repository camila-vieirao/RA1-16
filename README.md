## Execução
1. Clone o repositório:
```bash
git clone https://github.com/camila-vieirao/RA1-16.git
```
2. Navegue até a pasta do projeto:
``` bash
cd RA1-16
```

3. Para rodar o programa, execute o comando: 
``` bash
python ra1.py <arquivo-de-teste>.txt
```
